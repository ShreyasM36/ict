Product

My final product will be a virtual assistant powered by artificial intelligence, targeted towards helping students with stress relief and organisation. The virtual assistant will be hosted on a website which can be accessed by anyone. The graphical output of the website will be designed using HTML and CSS while the main code for the virtual assistant will be programmed using Javascript. The assistant will be able to communicate and have basic conversations with a user. Study tools like an in-built word descriptor, calculator, deadline reminder and chill study music player will be included to decrease overall stress and aid with organisation. The virtual assistant will receive audio input by the user, use speech recognition to convert the audio input to text, analyse the text and use text to speech to come up with a suitable response or action.
